# DormitoryReservationSystem

**สร้าง venv**
`python3 -m venv ชื่อ`

**run venv**
`.\ชื่อ\scripts\activate`

**ติดตั้ง**
`pip install -r requirements.txt`

**อัพเดตฐานข้อมูล**
`python manage.py migrate`

**สร้าง user admin**
`python mamage.py createsuperuser`

**รันเซิฟเวอร์**
`python manage.py runserver`
